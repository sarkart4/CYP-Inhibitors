        i. extract raw data from DTC, Excape, ChEMBL (DTC, Excape in sane notebook, ChEMBL seperate), 
       ii. apply standard AMPL data curation functions
      iii. create union set
2. Data Analysis
3. Deploy Model
4. HyperParameter Optimization
5. Model Evaluation
